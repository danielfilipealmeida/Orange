# Orange
Simple VJing app made with openFrameworks

**Orange** is the remake of **Carob**, a VJing software using *openFrameworks*.

## Configuration

To compile:
 - clone this inside: `<of path>/apps/myApps/`.
 - configure the following paths:
   - CATCHCPP_PATH
   - OF_PATH 


## Work
